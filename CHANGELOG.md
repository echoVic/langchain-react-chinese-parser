## 贡献指南

欢迎提交 Issue 和 Pull Request！

### 版本发布流程
1. 更新 CHANGELOG.md
2. 更新 package.json 版本号
3. 运行测试套件
4. 创建 Git Tag
5. 发布到 npm

### 语义化版本说明
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

感谢所有贡献者的努力！ 🙏
